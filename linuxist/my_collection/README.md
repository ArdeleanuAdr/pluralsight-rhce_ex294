# Ansible Collection - linuxist.my_collection

Documentation for the collection.

COMMANDS USED TO INITIALIZE A Collection in Ansible Galaxy
You then went inside of the collection and created a role with ansible-galaxy role init

  ansible-galaxy collection list
  ansible-galaxy collection init linuxist.my_collection
  cd my_collection/roles/
  ansible-galaxy role init chrony
  ansible-galaxy collection build .  ## dot (.) for local directory
  ansible-galaxy collection install linuxist-my_collection-1.0.1.tar.gz --upgrade
  ansible-galaxy collection list
